# Internship Portal Demo

A simple **Internship Portal Web Application** built using **HTML and CSS**.  
This project demonstrates a basic platform where students can explore internships and submit applications.

## 🚀 Features

- Responsive internship portal UI
- Internship listings with company details
- Application form for students
- Simple and clean interface
- Beginner-friendly frontend project

## 📌 Project Sections

The website contains the following sections:

### Home
Landing section introducing the internship portal.

### Internship Listings
Displays available internships such as:
- Web Development Intern
- Data Science Intern
- UI/UX Design Intern

Each listing shows:
- Company name
- Internship duration
- Apply button

### Application Form
Students can submit their details including:
- Name
- Email
- College
- Reason for applying

### Contact Section
Provides contact information for communication.

## 🛠️ Technologies Used

- HTML5
- CSS3
